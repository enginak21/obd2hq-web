# OBD2HQ Link Asset Plan - 2026-07-22

## Live Assets

- `/en/resources` and localized equivalents: central resource hub for linkable SEO assets.
- `/open-data/obd2-codes.json`: public OBD2 code dataset with attribution guidance.
- `/widget/obd2hq-lookup.js`: embeddable lookup widget for forums, blogs and repair resources.
- `/en/codes/{code}` plus localized code hubs: canonical generic code intent pages for GSC quick-win queries.
- Brand warning-light hubs such as `/en/cadillac/warning-lights`: target make-level warning light queries.

## Outreach Rules

- Use brand or naked URL anchors most of the time: `OBD2HQ`, `obd2hq.com`, `OBD2HQ OBD2 code guide`.
- Avoid paid dofollow packages, automated comments, exact-match anchor spam and copied guest posts.
- Prioritize genuine placements: automotive forums, GitHub resource lists, repair education pages, journalist source requests, YouTube descriptions and independent mechanic blogs.

## First Targets

- Promote the dataset to GitHub automotive/open-data lists.
- Offer the widget to small repair blogs and forum moderators.
- Use GSC queries `p0213`, `p0235 suzuki`, `p0251 ford`, `cadillac warning lights` as the first internal-link and outreach themes.
