# OBD2HQ Open Data

This folder documents the public diagnostic datasets published by OBD2HQ.

Canonical live resources:

- [OBD2 code dataset](https://obd2hq.com/open-data/obd2-codes.json)
- [Dashboard warning light dataset](https://obd2hq.com/open-data/warning-lights.json)
- [Diagnostic first-check checklist](https://obd2hq.com/open-data/diagnostic-checklist.json)
- [Linkable diagnostic assets](https://obd2hq.com/open-data/link-assets.json)

Main resource hub:

- [Free OBD2 resources for diagnostics and research](https://obd2hq.com/en/resources)

## Use Cases

- Automotive forums can cite a code guide or warning-light reference in real troubleshooting answers.
- Repair blogs can embed the lookup widget or cite the first-check checklist.
- Developers can use the JSON datasets for lookup tools, educational projects, and diagnostic apps.
- Driver education pages can cite the warning-light dataset for symbol explanations.
